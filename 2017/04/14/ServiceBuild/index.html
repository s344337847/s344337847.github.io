毕业设计啊！——服务器搭建篇

发表于 2017-04-14   |  

#系统环境
Ubuntu 16.04.1 LTS
服务器是腾讯云的体验版T_T，一核一G，非常吃力。

#开始配置
由于毕业设计的Server端是用java实现APP接口，所以服务器需要配置java、tomcat、mysql。

###配置java环境
apt install openjdk-8-jre-headless
这里安装的是openjdk 1.8版本，Openjdk与其他jdk的区别可以百度。
输入java -version出现版本号就表示安装成功了

###安装mysql
1、apt install mysql-server
2、apt-get isntall mysql-client
3、sudo apt-get install libmysqlclient-dev

安装的时候会提示设置密码

4、登录MySQLmysql -u root -p输入设置的密码就可以进入

安装好mysql后我们是不能远程连接的，我们需要设定防火墙和指定账户远程连接的权限

我们先授权账户能进行远程连接：

1、grant all privileges on *.* to root@"%" identified by "密码" with grant option;

参数 解释
*.* 第一个*代表数据库名，第二个*代表表名，*表示不限制只能访问的数据库或者表名
root@"%" root是授权账户，%是IP地址，%表示不限制IP地址
"password" 分配账号对应的密码，这里密码自己替换成你的mysql root帐号密码。

2、注意要刷新权限信息flush privileges;

3、开放3306端口，MySQL默认是3306端口

使用命令 ufw allow 3306/tcp 就可以了。使用ufw status可以查看防火墙状态

4、开放了端口，指定账户远程连接的权限还需要取消127.0.0.1IP绑定
vim /etc/MySQL/my.cnf（路径见安装目录，版本不一样可能不同）找到bind-address = 127.0.0.1
注释掉这行，如：#bind-address = 127.0.0.1
或者改为：bind-address = 0.0.0.0 允许任意IP访问，或者自己指定一个IP地址。

这样远程就能访问了！

###安装tomcat
apt install tomcat8
这里安装的是tomcat8版本，安装完之后访问http://localhost:8080出现It works !表示安装成功了
到这里就算是配置成功了，其它还要用到的到时在配置。
